# UI Guidelines

## Design Philosophy

- Clean
- Minimal
- Corporate
- Fast navigation

---

## Layout

Sidebar + Header + Content

Desktop:

- Fixed sidebar
- Sticky header

Mobile:

- Drawer sidebar

---

## Tables

Use tables for:

- Employees
- Assets
- Requests
- Departments

Features:

- Search
- Sorting
- Pagination
- Filters

---

## Forms

Use React Hook Form.

Requirements:

- Inline validation
- Error messages
- Loading states

---

## Status Badges

Employee Status:

- WORKING
- ON_NOTICE
- RESIGNED
- TERMINATED

Asset Status:

- AVAILABLE
- ALLOCATED
- MAINTENANCE
- TRASHED

Request Status:

- PENDING
- APPROVED
- REJECTED
- COMPLETED
